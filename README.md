# Enigma Encryption/Decryption Program

This is a simple C program that demonstrates the basic principles of the Enigma encryption machine. The Enigma machine was historically used by Germany during World War II for secure communication.

## Overview

The program allows you to perform two main operations: encryption and decryption. It uses a simplified version of the Enigma machine with three rotors. You can input a message and choose to encrypt or decrypt it.

## Prerequisites

Before you can use this program, you need to have a C compiler installed on your system.
